- 服务端渲染SSR（Server Side Rendering）

指将单页应用（SPA）在服务器端渲染成 HTML 片段，发送到浏览器，然后交由浏览器为其绑定状态与事件，成为完全可交互页面的过程


![](https://raw.githubusercontent.com/chengzao/imgbed/img/img/Server-Side-Rendering.jpeg)
